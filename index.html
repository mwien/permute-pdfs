Frequently Asked Questions

What are some potential use cases of this website?

You could can create random orders for questionaires in studies (medicine/psychology) or for taks in an exam. Maybe you can come up with other use cases, in that case feel free to write me.

What about privacy?

All your data (pdfs) is completely private and does NOT leave your device.

I need a slightly different feature than what is offered here. Can this be added?

Yes, please open an issue under TODO. The same holds if you encounter other problems.

How are the random permutations generated?

This is done in JavaScript using pseudo-random generators, more precisely the shuffle function provided by underscore (TODO: link). It should be sufficiently random for most applications and behave similarly to randomly sampled permutations in other languages such as R, Python etc.

Is it possible to seed the random generator in order to reproduce permutations?

No, this is unfortunately not possible at the moment, but may be added later.
